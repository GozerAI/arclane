### Logo Description (for designers)
The Arclane mark is a geometric letter "A" rendered inside a dark navy rounded square (#07111c, 7px radius). The "A" uses the brand gradient (blue-to-green, #4bb2ff to #38d39f). The letterform is sharp and architectural — clean cuts, no curves on the legs, with a visible crossbar. The overall feel is technical, modern, and minimal.

The wordmark "arclane" is set in the primary font at weight 800 (extra bold), lowercase, with tight letter-spacing (-0.5px). On dark backgrounds it's #edf4ff (near-white). On light backgrounds it's #07111c (navy).

### Logo Usage Rules
- Minimum clear space: half the height of the mark on all sides
- Minimum size: 24px height for mark only, 100px width for full logo
- Never rotate, stretch, or add effects
- Always use provided SVGs — don't recreate from text
